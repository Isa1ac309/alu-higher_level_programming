# Python - More Classes and Objects

Deeper object-oriented programming in Python: properties, magic methods,
class attributes, static methods, and class methods, built up through a
`Rectangle` class.

## Learning objectives

- Why and how to use properties instead of getters and setters
- What `__str__` and `__repr__` are and how they differ
- What `__del__` does and when it is called
- The difference between a class attribute and an instance attribute
- What a static method and a class method are, and when to use each
- What a docstring is and why every module, class, and method needs one
