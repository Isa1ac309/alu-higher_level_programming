# Python - Everything is object

An exploration of how Python treats objects: identity, type, mutability,
aliasing, and how arguments are passed to functions.

## Learning objectives

- What an object is and the difference between a class and an object
- The difference between a reference, an assignment, and an alias
- How to know if two variables are identical or linked to the same object
- What mutable and immutable objects are, and which built-in types are which
- How Python passes variables to functions

## Files

Answer files (`*-answer.txt`) contain the one-line answer to each question.

| File | Description |
|---|---|
| `0-answer.txt` to `18-answer.txt` | Identity, equality, and mutation questions |
| `19-copy_list.py` | Function that returns a copy of a list |
| `20-answer.txt` to `28-answer.txt` | Tuple and mutation questions |

## Requirements

- Ubuntu 20.04 LTS, Python 3.8.5
- All files end with a new line
- Code follows pycodestyle 2.7.*
- No modules may be imported
