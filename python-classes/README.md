# Python - Classes and Objects

Introduction to object-oriented programming in Python: classes, instances,
attributes, encapsulation, properties, and data abstraction.

## Learning objectives

- What OOP is and what a class, an object, and an instance are
- The difference between a class attribute and an instance attribute
- What `self` is and how attribute lookup works
- Why and how to make an attribute private with name mangling
- How to use properties (getters and setters) to validate assignment
- What a docstring is and why every module, class, and method needs one

## Files

| File | Description |
|---|---|
| `0-square.py` | Empty class Square |
| `1-square.py` | Square with a private size attribute |
| `2-square.py` | Adds type and value validation on size |
